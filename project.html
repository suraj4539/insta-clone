PROJECTS:

Instagram Clone

The "Instagram Clone" project is a static profile page displaying user details, a photo grid, and placeholders for likes and comments. It focuses on building a clean, responsive UI using HTML, CSS, and JavaScript, without backend or authentication features, offering an introductory experience in frontend development.

Dessert~Nest

The interface showcases a variety of desserts, such as donuts, cakes, and brownies, with each item featuring an image, description, and "Add to Cart" button. A navigation bar offers options like Home, Categories, Orders, and Contact. The clean, visually appealing design emphasizes product presentation, making it ideal for a bakery's online shop.

Super Mario Blog

"Super Mario Blog" is a fan site dedicated to the Super Mario franchise, featuring articles, game updates, tips, and trivia. It has a colorful, retro design with interactive elements like comments and image galleries, offering fans a fun space to explore and discuss the series.
